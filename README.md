# Homebrew Tap for dotbrains

This is the official Homebrew tap for dotbrains tools.

## Installation

```bash
# Add the tap
brew tap dotbrains/tap
```

### ConfigSync

```bash
brew install --cask dotbrains/tap/configsync
```

### PRR

```bash
brew install --cask dotbrains/tap/prr
```

### AWSW

```bash
brew install --cask dotbrains/tap/awsw
```

## Available Casks

- **configsync** - Synchronize macOS application configurations across machines
- **prr** - AI-powered PR code review CLI
- **awsw** - Seamlessly switch between AWS SSO accounts and EKS contexts

## Contributing

This repository is automatically maintained. Cask updates are handled by GoReleaser in the respective tool repositories.

## Support

For issues with specific tools, please file issues in their respective repositories:
